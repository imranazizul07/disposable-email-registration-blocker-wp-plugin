Blocks user registration using temporary disposable emails in all registration options including Ultimate Member, WooCommerce, WP Default login, and any plugin using the user_register hook.

== Description ==

The Block Disposable Emails plugin prevents users from registering with temporary disposable email addresses across various registration forms including:

- WordPress default registration
- Ultimate Member registration
- WooCommerce registration
- Any custom registration forms using the user_register hook

It helps in maintaining the integrity of your user base by blocking spam registrations and ensuring that only valid email addresses are used.

**Key Features:**
- Blocks disposable email domains during registration.
- Supports WordPress default, Ultimate Member, and WooCommerce registrations.
- Customizable list of disposable email domains.
- Automatically pull the latest list of disposable email domains from an online source.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/block-disposable-emails` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'Disposable Emails' screen to configure the plugin settings.

== Frequently Asked Questions ==

= How does the plugin block disposable emails? =

The plugin checks the email domain against a list of known disposable email domains during the registration process. If a match is found, the registration is blocked.

= Can I add my own disposable email domains? =

Yes, you can add or remove email domains from the plugin settings page.

= How can I update the list of disposable email domains? =

You can pull the latest list of disposable email domains from an online source directly from the plugin settings page.
